Trabalho final de EC108 - Logistica Sanoj

You'll find below the operations and it's descriptions in both SOA and REST.

In order to access the operations, you'll have to use an address like this: http://localhost:8080/LogisticaJonas + operationName/{parameter - if exists} after you deploy the project.

SOAP Operations

  • getDeliveries - Returns all deliveries.
  • getDeliveriesToDeliver - Returns the deliveries to be delivered.
  • getDeliveryById - Returns the deliveries by ID.
  • getDeliveryByOrderId - Returns the delivery by Sales Order ID.
  • insertDelivery - Add a new delivery with Sales Order ID and CRM Client ID.
  • makeDelivery - Register a delivery performed with the delivery informations.
  • deleteDelivery - Delete a delivery from the database.

Database Description

This is the database Delivery, that is used in this project.

CREATE TABLE IF NOT EXISTS `delivery` (
`id` bigint(20) NOT NULL AUTO_INCREMENT,
`orderID` bigint(20) NOT NULL,
`clientID` bigint(20) NOT NULL,
`rName` varchar(100) DEFAULT NULL,
`rCPF` varchar(12) DEFAULT NULL,
`isClient` tinyint(1) DEFAULT NULL,
`deliveredTime` timestamp NULL DEFAULT NULL,
`deliveredLatitude` varchar(60) DEFAULT NULL,
`deliveredLongitude` varchar(60) DEFAULT NULL,
`status` varchar(50) DEFAULT NULL,
PRIMARY KEY (`id`)
)
